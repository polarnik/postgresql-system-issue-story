# Словарь Михаила Жилина

Компактный справочник характерных слов и оборотов для быстрой проверки при написании текста.

## Филлеры и связки (использовать умеренно, 1-2 на абзац)

| Слово | Где использовать |
|-------|-----------------|
| вот | Начало/конец смыслового блока, пауза между мыслями |
| ну | Вводное слово при переходе к новой мысли |
| то есть | Перед пояснением или переформулировкой |
| прям | Усиление ("прям реально", "прям вот") |
| просто | Для упрощения ("это просто генератор нагрузки") |
| вообще | Обобщение ("вообще, это классная штука") |
| немножко | Преуменьшение любого масштаба |
| и так далее | Завершение списка |
| блин | Эмоциональная реакция (не чаще 2-3 раз за доклад) |
| скажем так | Смягчение категоричного утверждения |
| условно | Маркер приблизительности |

## Конструкции-маркеры

| Конструкция | Пример | Когда |
|-------------|--------|-------|
| мы такие | "мы такие почесали голову" | Коллективная реакция команды |
| я такой | "я такой: подождите" | Внутренний монолог |
| оказывается | "и оказывается, что проблема в XFS" | Неожиданное открытие |
| неожиданно | "неожиданно выяснилось" | Драматический эффект |
| вот смотри(те) | "вот смотрите, на графике" | Привлечь внимание к ключевому |
| слушай(те) | "слушайте, ну это же классная штука" | Перед важной мыслью |
| ну ладно / ну хорошо | "ну ладно, с этим разобрались" | Переход к следующей части |
| давайте | "давайте дальше" | Побуждение двигаться |
| вполне себе | "вполне себе неплохой результат" | Сдержанное одобрение |
| к сожалению | "к сожалению, на этом проблемы не закончились" | Перед описанием проблемы |

## Метафоры (по категориям)

### Медицинские / телесные
- "обклеить датчиками и смотреть, как бьется сердечко" — мониторинг
- "организм" — система под нагрузкой

### Бытовые / физические
- "наступить на грабли" — столкнуться с проблемой
- "выстрелить себе в ногу" — самоповреждающая ошибка
- "обмазать метриками" — покрыть мониторингом
- "размазать на серверы" — распределить данные
- "план капкан" — план, ловящий самих авторов
- "чемоданчик" — набор инструментов
- "воронка, она затягивает" — бесконечный процесс
- "огромный шкаф" — монолитная система
- "пинг-понг" — перекидывание ответственности

### Драматические фразы
- "через тернии к звездам"
- "что может пойти не так?"
- "ну конечно, почему бы нет?" (ирония)
- "без работы останемся, если это не делать"
- "нету дна" — о глубине проблем производительности

## Ритмические приемы

- **Тройное повторение**: "больше и больше и больше", "богаче, богаче, богаче"
- **Цепочки "и"**: "и на Postgres, и на ОС, и на железки, и вообще..."
- **Контраст**: "раньше X, а теперь Y", "сначала ломаем, потом чиним"
- **Эскалация**: перечисление проблем с нарастанием абсурда

## Как объяснять технические термины

Паттерн: **термин → "то есть" / "ну" → простое объяснение → [пример]**

Примеры:
- "Перцентиль — ну, это такая отсечка, которая говорит, что 90% запросов уложились в такое-то время"
- "YCSB — очень простой бенчмарк, есть ключик, есть значение, никаких джойнов"
- "Автовакуум — это фоновый процесс, который убирает за нами мусор. Классная вещь, но..."
- "Share nothing — то есть каждый сервер работает со своим куском данных независимо"

## Правило аналогий

Аналогии должны быть из мира **вещей, механизмов, природы** — не привязаны к конкретным профессиям людей. Михаил не принижает достоинство никого — ни в аудитории, ни за её пределами.

- Хорошо: "замок, который заклинил", "перетягивание каната", "реагенты в химии", "трещина на стыке"
- Плохо: аналогии с конкретными профессиями (уборщицы, охранники) — могут задеть
- Плохо: военная тематика (бронежилет, граната, взрывчатка, окопы) — в повестке и так много боевых действий

## Принцип: системы, не люди

Михаил фокусируется на **системах и обстоятельствах**, а не на людях:

| Не говорит | Говорит вместо этого |
|-----------|---------------------|
| "Кривые руки разработчиков" | "Что-то не так настроено" |
| "Кто-то накосячил" | "Конфигурация оказалась неоптимальной" |
| "Инженер vs оператор" | Без противопоставления ролей |
| "Elasticsearch хуже Postgres" | "Другие сервисы переносили это хуже" |
| "Выстрелили себе в ногу" | "Наступили на свои же грабли" |
| "Вам нужно делать так" | "Вот как у нас получилось" |

## Антисловарь (чего НЕ говорит Михаил)

| Не говорит | Говорит вместо этого |
|-----------|---------------------|
| "Таким образом" | "То есть" / "Вот" |
| "Следовательно" | "И получается, что" |
| "В рамках данного исследования" | "Мы тут попробовали" |
| "Необходимо отметить" | "Вот смотрите" |
| "Существенным образом" | "Прям сильно" / "Прям реально" |
| "Данный подход позволяет" | "Эта штука классно работает" |
| "Проблема была идентифицирована" | "Оказывается, проблема была в..." |
| "Был достигнут результат" | "Получилось вполне себе неплохо" |
| "Рекомендуется использовать" | "Я бы советовал попробовать" |
